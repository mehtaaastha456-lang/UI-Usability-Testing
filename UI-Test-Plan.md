# UI & Usability Test Plan – Amazon Website

## Objective
The objective of this test is to evaluate the user interface and usability of the Amazon website.

## Scope
The testing will cover the following areas:

- UI consistency
- Navigation usability
- Page responsiveness
- Accessibility
- Cross-device compatibility

## Modules Covered

- Home Page
- Login Page
- Product Search
- Product Page
- Cart Page

## Testing Types

- UI Testing
- Usability Testing
- Accessibility Testing
- Responsive Testing

## Test Environment

Browser: Google Chrome  
Operating System: Windows 10  

## Entry Criteria

- Website should be accessible
- Test scenarios prepared

## Exit Criteria

- All UI issues documented
- Usability observations completed

## Deliverables

- UI Test Plan
- UI Test Scenarios
- Screenshots
- Usability Observations
